# UK / NZ / Canada — Short Pathway References

> These are stubs. For full planning, always cross-reference

## 🇬🇧 United Kingdom

### Student → work → PR

```
Student visa (Tier 4 → now "Student Route")
  → Graduate Route (2 yrs bachelor / master, 3 yrs PhD) — no sponsor
  → Skilled Worker visa (points-based, employer-sponsored)
  → ILR (Indefinite Leave to Remain) after 5 yrs continuous residence
  → Citizenship after 1 yr ILR + Life in UK test
```

### Skilled Worker points

Need 70 points total:
- 50 mandatory: job offer from licensed sponsor + appropriate skill level + English (CEFR B1+)
- 20 tradeable: salary over the going rate, shortage occupation, PhD in relevant field, etc.

### Salary thresholds (verify on gov.uk)

| Tier | Salary (general) | Shortage / new entrant |
|---|---|---|
| General Skilled Worker | £38,700 (or job's going rate) | £30,960 |
| Healthcare / education | Lower bands apply | — |

### Useful angles for the user

- Russell Group universities = research prestige (not all UK universities are equal)
- Graduate Route is the easiest post-study work visa; budget for it
- Scotland has separate occupation shortage lists via ScotGov

## 🇳🇿 New Zealand

### Student → work → PR

```
Student visa
  → Post-study work visa (1-3 yrs depending on qualification + location)
  → Skilled Migrant Category (SMC) EOI
  → Resident Visa
```

### SMC points (key ones)

| Category | Max |
|---|---|
| Skilled employment (offer) | 50 |
| Work experience (in NZ, skilled) | 30 |
| Qualifications | 70 (PhD = 70; Master = 50; Bachelor = 40) |
| Age (25-29) | 30 |
| Partner bonus | 20 |

Current threshold (verify on ird.govt.nz): typically 160 points to be
selected from the pool.

### Green List (fast-track occupations)

NZ's Green List lets certain occupations go straight to a Straight
to Residence visa or Work to Residence visa. If the user's occupation
is on the Green List, the pathway is much faster.

## 🇨🇦 Canada

### Student → work → PR

```
Study permit
  → PGWP (Post-Graduation Work Permit, up to 3 yrs)
  → Canadian work experience (1+ yr in NOC 0/A/B)
  → Express Entry (CEC — Canadian Experience Class) OR PNP
  → PR
```

### Express Entry — 3 streams

| Stream | Best for |
|---|---|
| Federal Skilled Worker (FSW) | Skilled workers abroad with foreign work experience |
| Canadian Experience Class (CEC) | People who already worked in Canada on PGWP |
| Federal Skilled Trades (FST) | Trade occupations |

### CRS — Comprehensive Ranking System (top categories)

| Factor | Max |
|---|---|
| Age | 110 (sweet spot 20-29) |
| Education | 150 (PhD top) |
| Language (EN + FR) | 160 |
| Canadian work experience | 80 |
| Skill transferability factors | 100 |
| Additional (PNP, sibling, French) | 600 |

### Provincial Nominee Programs (PNP)

Almost every province has its own. Some are aligned with Express Entry
(+600 CRS), some are base-level. If the user has flexibility on where
to settle in Canada, PNP is the easiest back-up to federal Express Entry.
