## Visa subclasses (the four you care about)

| Subclass | Name | Sponsorship | PR path? | Best for |
|---|---|---|---|---|
| 500 | Student visa | Education provider | No (temporary) | Studying in Australia |
| 485 | Temporary Graduate | None | No (temporary, 18m-4y) | Work after graduation |
| 189 | Skilled Independent | None — points-tested | Yes — direct PR | High scorers, no state tie |
| 190 | State Nominated | A state/territory | Yes — direct PR | 5 extra points, state willing to nominate you |
| 491 | Skilled Work Regional (Provisional) | State/territory or family | No (leads to 191 → PR after 3y) | Regional living for 15 extra points |
| 191 | Permanent Residence (Regional) | n/a (follow-on from 491) | Yes | Settled regional workers |

The user's pathway is usually: 500 → graduate → 485 → EOI in SkillSelect
→ 189 or 190 (or 491 → 191).

## EOI points table (SkillSelect, points-tested visas)

Points add up across the categories below. Minimum to lodge an EOI is 65;
the **competitive** score for most occupations in 2025-2026 is much higher.

| Category | Maximum | Sub-options |
|---|---|---|
| Age (at invitation) | 30 | 25-32 yrs = 30; 33 = 25; 39 = 15; 45+ = 0 |
| English | 20 | Competent (IELTS 6) = 0; Proficient (7) = 10; Superior (8) = 20 |
| Qualifications | 20 | Bachelor / Master = 15; Doctorate = 20 |
| Work experience (overseas, in nominated occupation) | 15 | 3-5y = 5; 5-8y = 10; 8+y = 15 |
| Work experience (Australian, in nominated occupation) | 20 | 1-3y = 5; 3-5y = 10; 5-8y = 15; 8+y = 20 |
| Specialist education (STEM PhD / professional year) | 10 | Australian Masters by research / PhD in STEM = 10 |
| Other factors (Credentialled community language, partner skills, state nomination, regional study) | 15-25 | State nomination = 5; Regional study = 5; Partner skills = 5-10 |

**Australian study bonus (5 pts):** requires 2 academic years of a
CRICOS-registered course in Australia that the user studied in person.
This stacks with regional study (5 pts) only if the regional study was
in a different period.

## English test mapping (Home Affairs)

| Test | Competent (0 pts) | Proficient (10 pts) | Superior (20 pts) |
|---|---|---|---|
| IELTS | 6.0 each band | 7.0 each band | 8.0 each band |
| PTE Academic | 50 each band | 65 each band | 79 each band |
| TOEFL iBT | 64 (R/L 13, S 18, W 21) | 94 (R/L 24, S 23, W 27) | n/a (use PTE/IELTS) |
| CAE | 169 each band | 185 each band | 200 each band |

**Note:** PTE and IELTS are usually interchangeable for entry to
university, but Home Affairs only accepts certain tests for visa.
Always recheck on the Home Affairs page for the visa subclass being
applied for.

## State / territory nomination rules (high level)

States set their own occupation lists, criteria, and processing times.
They can pause, reopen, or change priorities monthly.

| State | Common angles |
|---|---|
| NSW | High thresholds; usually needs 90+ pts; Sydney heavy |
| VIC | Healthcare and education priority; otherwise limited |
| QLD | 190 + 491 both active; QLD 491 often easier than 190 |
| SA | Active; well-known 491 pathway |
| WA | Building and healthcare priority; separate WA list |
| TAS | Long-standing 491 pathway; "genuine intention to live" test |
| ACT | Canberra matrix — points-based inside the state |
| NT | Darwin — often the most accessible for regional + lower points |

Always check the state government's own occupation list (not Home Affairs)
for the live "is my occupation open?" answer.

## Course quality & employment data (QILT)

QILT = Quality Indicators for Learning and Teaching. Public, free data
on every Australian higher-education course. Two tables matter:

| Table | URL | What it tells you |
|---|---|---|
| Course Experience Questionnaire (CEQ) | https://www.qilt.edu.au/survey-(CEQ) | Graduate satisfaction, skill development |
| Graduate Outcomes Survey (GOS) — Employment | https://www.qilt.edu.au/survey-(GOS) | Full-time employment rate + median salary by course + university |
| GOS — Overall Employment | same | Includes part-time + casual; broader than FT only |

When comparing two universities for the same course, GOS full-time
employment + median salary is the cleanest single metric. CEQ is more
about student experience than outcomes.

| Field | Assessing authority | Notes |
|---|---|---|
| Occupational Therapy | Occupational Therapy Council (Australia & NZ) Ltd (OTC) | Two-stage: assessment for migration + assessment of competence |
| Nursing / Midwifery | Australian Health Practitioner Regulation Agency (AHPRA) | Registration = skills assessment for many health roles |
| Engineering | Engineers Australia | Competency Demonstration Report (CDR) for non-Australian degrees |
| Accounting | CPA Australia / CA ANZ / IPA | One of three |
| ICT | Australian Computer Society (ACS) | Skills assessment based on qualification + work experience |
| Teaching | AITSL | Skills assessment + English higher than most visas |

Find the right body: https://immi.homeaffairs.gov.au/visas/working-in-australia/skill-occupation-list/assessing-authorities

## Quota trends (FY 2024-25, 2025-26)

The 189 federal quota for 2025-26 was cut from 30,375 to 16,900.
Most non-healthcare occupations saw their invitation cutoffs rise
into the 85-95 band. Healthcare (nursing, OT, physio, etc.) tends to
remain 75-80.

Always check the latest SkillSelect invitation round data — Home
Affairs publishes it after every round.

## Recommended reading order (when planning a pathway)

1. **Is the occupation on MLTSSL?** If no, stop. Either change occupation
   or change country.
2. **Skills assessment body for the occupation.** What evidence do they
   need? Does the user's degree meet it without extra supervised practice?
3. **University entry requirements.** English, GPA, prerequisites.
   Recheck against the user's actual scores.
4. **EOI points calculation.** Use the table above. Compute base + 5 for
   state nomination if applicable.
5. **State nomination list for the user's occupation.** Even if 189 is
   the goal, a state 190 or 491 can be a useful back-up.
6. **QILT data for the shortlist.** Employment rate + salary tells the
   user the realistic outcome.
