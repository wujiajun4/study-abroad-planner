## When a source is missing

If the user asks about a country not in this file:
1. Be honest — say you don't have a reference for it.
2. Ask the user to confirm the country and target field, then offer to
   research it inline with public sources.
3. If you do research, log every URL you use in the report so the user
   can verify. Do not paraphrase government pages without linking.
